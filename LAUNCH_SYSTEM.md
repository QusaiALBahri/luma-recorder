# Luma Recorder Launch System

Luma Recorder is positioned as:

```text
Record. Explain. Teach.
```

A local-first screen recorder built specifically for educators, trainers, and knowledge creators.

## Launch Files

- [README.md](README.md): product landing page
- [LAUNCH_STRATEGY.md](LAUNCH_STRATEGY.md): external traffic and launch timeline
- [CONTENT_PACK.md](CONTENT_PACK.md): launch posts for LinkedIn, Twitter/X, Reddit, Hacker News, and Dev.to
- [COMMUNITY_PLAN.md](COMMUNITY_PLAN.md): community growth, contribution hooks, and GitHub growth mechanics
- [ASSET_PLAN.md](ASSET_PLAN.md): screenshots, GIFs, demo video, SEO, and topic plan

## Launch Goal

Convert educators, trainers, and knowledge creators into:

- users
- GitHub stars
- feedback contributors
- translators
- workflow contributors

If Luma Recorder helps you record lessons or training videos, star the repo so more educators can discover it.
