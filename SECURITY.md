# Security Policy

## Official Source

The official source is published by **albahri.org**.

- <https://github.com/albahri-org>
- <https://github.com/QusaiALBahri?tab=repositories>

Download Luma Recorder only from the official source or a release linked from the official source.

## Local-First Design

Luma Recorder is designed as a portable local Windows app.

- No telemetry
- No account sign-in
- No cloud sync
- No upload feature
- No background service
- No network feature

## Reporting Security Issues

Please report security issues privately to the project maintainer before public disclosure.

Include:

- App version
- Windows version
- Reproduction steps
- Expected result
- Actual result
- Screenshots or logs if useful

## Protected Content

Admin mode is intended for recording normal elevated applications owned or controlled by the user. It does not bypass Windows secure desktops, login screens, DRM/protected video, or operating-system privacy protections.
